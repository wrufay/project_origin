# ProjectOrigin

Point your camera at an object, get its Chinese name and pronunciation — powered by Claude.

## Run it

**1. Backend**
```bash
cd backend
npm install
cp .env.example .env   # add your ANTHROPIC_API_KEY
npm run dev
```

**2. App** (new terminal, from the repo root)
```bash
npm install
npx expo start
```

Scan the QR code with Expo Go, or press `i` / `a` / `w` for simulator/emulator/web.
